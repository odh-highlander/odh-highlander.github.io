# Open Data Hub - Highlander

![ODH+EasyBuild+Lmod](./img/banner.png)

Open Data Hub Highlander brings the power of Linux environments modules to Data Science platforms running on Kubernetes!

The reference architecture and implementation is based on [Open Data Hub](http://opendatahub.io/), the Data and AI platform running on [OpenShift](https://www.redhat.com/en/technologies/cloud-computing/openshift).
